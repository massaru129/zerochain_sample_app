# zerochain_sample_app

Rustで定義した関数をKotlin経由でFlutterから呼び出すことをテストするアプリ

見るべきコードは下記３つのみ

* `rust/src/lib.rs`
* `android/app/src/main/java/sample/example/com/zerochain_sample_app/MainActivity.kt`
* `lib/main.dart`
